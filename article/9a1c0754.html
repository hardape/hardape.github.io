先导例子

语法

table:声明一个表格

属性 说明
align 表格相对周围元素的对齐方式，left/ center/right
border 表格是否有边框，1 / "",默认为"",表示无边框
width 设置表格的宽度
height 设置表格的高度
cellpadding 设置内容和边框的距离，默认1像素
cellspacing 设置边框的大小,即单元格的间距，默认2像素

tr:声明一行

属性 说明
height 设置行高,即设置该行所有单元格的高度
rowspan 行合并
colspan 列合并

th / td:声明一个单元格

  • th:表头格,默认居中加黑显示数据

  • td:普通单元格,默认居左显示数据

属性 说明
width 设置单元格的宽度,即设置列宽

注意

  • 行高即该行所有单元格的宽度.
  • 单元格的宽度即列宽.

表格结构标签

标签名 说明
thead 表格的头，用于定义表格的头部
tbody 表格的主体，主要用于放置数据本体
tfoot 表格的脚注

表格的合并

  1. 首先确保表格是一个规整的表格
  2. 根据要合并的单元格,找到其所在的源码位置
  3. 行合并:在要合并的单元格中的上侧单元格上使用属性rowspan="要合并的单元格的个数",并删除其他要合并的单元格完成合并.
  4. 列合并:在要合并的单元格中的左侧单元格上使用属性colspan="要合并的单元格个数",并删除其他要合并的单元格完成合并.
