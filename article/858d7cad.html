标签属性

HTML 标签可以拥有属性。属性提供了有关 HTML 元素的更多的信息。

注意

  1. 一个HTML标签可以拥有多个属性，属性必须写在标签名的后面。
  2. 属性之间不分先后顺序，标签名与属性、属性与属性之间均以空格分开。
  3. 属性采用键值对的格式，即 key = “value” 的格式，属性=“属性值”

列表标签

无序列表

使用

li标签中书写列表内容,一个li标签代表列表中的一行数据

特点

默认数据前有一个黑圆圈符号，可通过css修改样式

注意

  1. ul标签只能嵌套li标签，不能在ul标签中输入其他标签或者文字
  2. li标签中可容纳任意元素

有序列表

使用

li标签中书写列表内容,一个li标签代表列表中的一行数据

特点

会自动地给列表进行顺序编码，格式从小到大并且是连续的.

属性

默认使用阿拉伯数字进行顺序编码

自定义列表

使用

dt : 数据的标题

dd : 数据的具体内容，一个dd标签表示一条数据

注意

自定义列表常用于对术语或名词进行解释和描述，列表项没有任何符号。

图片标签

属性

属性 作用 说明
src 图片路径 必须属性
alt 代替文本 规定在图像无法显示时的替代文本
title 图片标题 提示文本，鼠标放在图片上的时候显示
width 图片宽度 设置图片的宽度
height 图片高度 设置图片的高度
border 图片边框 设置图像的边框粗细，单位是像素

注意

1.图片标签不会自动换行(行内元素)

2.如果是单独设置图片的宽/高，则在图片保证不失真的情况下自动缩小或放大，即保持纵横比等比例缩放；单位可以是px(像素)或%(百分比)

3.图片路径(可以使用本地资源/网络资源)

  • 相对路径:从当前文件夹出发查找另一个文件所经过的路径

    相对路径分类 符号 说明
    同一级路径 图像文件位于HTML文件同一级，如<img src="baidu.jpg />"
    下一级路径 / 图像文件位于HTML文件下一级，如<img src="images/baidu.jpg />"
    上一级路径 …/ 图像文件位于HTML文件同一级，如<img src="../baidu.jpg />"

  • 绝对路径:从根盘符出发所查找文件的路径或网络URL地址
