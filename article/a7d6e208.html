09-HTML的表单&表单域标签

字数总计:1.4k|阅读时长: 5 分钟

表单标签

语法

作用

收集并提交用户数据给指定服务器

属性

属性 说明
action 收集的数据提交地址即URL
method get:适合小量数据,表单数据以？隔开拼接在URL后面，不同的键值对使用&符号隔开,不安全;post:适合大量数据，安全，隐式提交

注意

  1. 表单数据的提交,要提交的表单必须拥有name属性值,否则不会提交.提交的表单项数据为键值对,键为name属性的值,值为用户书写的数据(value值)
  2. form标签会收集其标签内部的数据
  3. form表单的数据提交需要依赖于submit提交按钮.

表单域标签(行内元素)

文本框

语法

属性

属性 说明
type="text" 表示该input标签为文本框，收集少量的文本数据,用户可见
name 数据提交的键,也会被js使用
value 默认值，要提交的数据值
maxlength 正整数，规定输入字段中的字符的最大长度，一般较少使用

密码框

语法

属性

属性 说明
type="password" 表示该input标签为密码框，收集少量的文本数据,用户不可见
name 数据提交的键,也会被js使用
value 默认值，要提交的数据值
maxlength 正整数，规定输入字段中的字符的最大长度，一般较少使用

单选框

语法

男 女

属性

属性 说明
type="radio" 表示该input标签为单选框
name 数据提交的键,name属性值相同的单选框只能选择一项数据
value 要提交的数据值
checked="checked" 使用此属性的单选框默认是选择状态

多(复)选框

语法

看电影 游泳 旅游

属性

属性 说明
type="checkbox" 表示该input标签为多(复)选框
name 数据提交的键,一个多选组需要使用相同的name属性值
value 要提交的数据值
checked="checked" 使用此属性的多选框默认是选择状态

label标签

定义

为input元素定义标注（标签）

语法

属性

属性 说明
for 属性值应当与相关元素的id属性值相同

作用

用于绑定一个表单域标签，当点击label标签内的文本时，浏览器就会自动将焦点（光标）转到或者选择对应的表单元素上用来增加用户体验。

单选下拉框

语法

属性

属性 说明
name 数据提交的键名

子标签

option : 一个option标签表示一个下拉选项

属性 说明
value 要提交的数据
selected="selected" 使用此属性的单选默认是选择状态

文本域

语法

作用

声明一个可以书写大量文字的文本区域

属性

属性 说明
name 数据提交的键名
rows 声明文本域的行数
cols 声明文本域的列数

注意

textarea语句无法跟input单标签一样在标签内向value赋值，可以和其他双标签语句一样,在双标签中间直接写入赋值，文本域标签需要写在一行中，否则内容会有空白区域
